# Vorlesung 1

### Klassen
"Konstrunktions anleitung" für Objekt \
führt Datentypen ein\
Objekt werden bei Bedarf aus der Klasse erzeugt **Beispiel:**
```
public class Main {
int x = 5;
public static void main(String[] args) {
    Main myObj = new Main();
    System.out.println(myObj.x);
  }
}
```
